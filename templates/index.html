OSI Problem

The OSI Problem deals with predicting shoppers intention, inorder to provide more customer friendly environment in the shopping site.I have collected the data from UCI Machine learning repository (You can collect it through here). This data conatins the information related the revenue generated by the customers. I have preprocessed the data and used K nearest neighbours KD tree algorithm to build the model. I have created new features combing the existing data and they have undergone teqnuiques such as Correlation and ExtraTreesClassifier. You can try it out by clicking the below button.

The Metrics Used

I have used 2 metrics inorder to confirm. I have used Accuracy as the one metric and F1 score as another metric. As you see the both Accuracy and F1 score decreases along with increasing the K value. F1 score provides kind of confidence that my model do predict well. Here Blue line coresponds to the Training Dataset and Orange coresponds to Validation Dataset.
